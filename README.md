check_browser
=============

Use javascript userAgent check browser

```javascript

alert(browser());
// return browser name IE or Chrome ...

// or
var b = new browser();
alert(b.safari);
// return true or false
```

### Method support browser

* edge
* opera
* firefox
* chrome
* safari
* ie7
* ie8
* ie9
* ie10
* ie11
* ie
